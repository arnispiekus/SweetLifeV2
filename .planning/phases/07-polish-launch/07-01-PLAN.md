# Phase 7 Plan 01: Framer Motion Animations

## Context
Phase 7 focus is polish and launch. Framer Motion is already installed (v12.27.0) but unused. Currently using CSS animations and Tailwind built-ins only. Goal: Add smooth, warm animations that match Sweet Life's inviting cafe brand.

## Tasks

### Task 1: Create motion utilities and wrapper components
Create reusable Framer Motion components for consistent animations:
- `FadeIn` - Fade in with optional vertical/horizontal slide
- `StaggerContainer` - Parent for staggered child animations
- `ScrollReveal` - Viewport-triggered reveal animations

Files: `src/components/motion/index.ts`, individual component files
Patterns: Use `motion` from framer-motion, `useInView` hook for scroll triggers

### Task 2: Add scroll reveal animations to Homepage sections
Apply `ScrollReveal` to homepage sections for gentle fade-in as user scrolls:
- Hero section (immediate, no delay)
- Specialties carousel section
- About preview section
- Reviews section
- Sushi section
- Private room section
- Location section

File: `src/app/page.tsx`
Approach: Wrap each section with ScrollReveal, use staggered delays for content within

### Task 3: Add animations to About page
Apply scroll reveals and stagger animations:
- Story section with image grid fade-ins
- Values cards with stagger effect
- Promise section

File: `src/app/about/page.tsx`

### Task 4: Add animations to Menu page
Subtle animations for menu browsing:
- Category cards with stagger reveal
- Menu items fade in on scroll
- Floating decorative elements gentle motion

Files: `src/app/menu/page.tsx`, `src/components/menu/MenuCategory.tsx`

### Task 5: Add animations to remaining pages
Apply consistent animations to:
- Contact page: form and map sections

Files: Various page files

Enhance interactive elements:
