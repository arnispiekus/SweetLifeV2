# Phase 7 Plan 02: SEO, Performance & Deployment

## Context
Animations complete (Plan 01). Now focus on SEO structured data, performance optimization, and Vercel deployment. Forms and mobile already work — this plan adds the finishing touches for launch.

## Tasks

### Task 1: Add LocalBusiness structured data (JSON-LD)
Create structured data component for rich search results:
- LocalBusiness schema with name, address, phone, hours
- Restaurant type with cuisine, menu URL, price range
- Add to root layout.tsx as script tag

File: Create `src/components/seo/StructuredData.tsx`, add to `src/app/layout.tsx`
Reference: https://schema.org/Restaurant

Schema content:
```json
{
  "@type": "Restaurant",
  "name": "Sweet Life Cafe",
  "address": {
    "@type": "PostalAddress",
    "streetAddress": "12 Monaghan Street",
    "addressLocality": "Newry",
    "postalCode": "BT35 6AA",
    "addressCountry": "GB"
  },
  "telephone": "+44 28 3044 8808",
  "url": "https://sweetlifecafe.co.uk",
  "servesCuisine": ["Korean", "Cafe", "Sushi"],
  "priceRange": "££"
}
```

### Task 2: Add OpenGraph images
Ensure OG images are set for social sharing:
- Add default OG image to public/ if not exists
- Add images property to root metadata
- Verify per-page OG tags are working

File: `src/app/layout.tsx`, check page metadata files

### Task 3: Run Lighthouse audit and fix critical issues
- Run `npm run build` to create production build
- Use Lighthouse CLI or browser DevTools on build
- Fix any critical performance issues (CLS, LCP, FID)
- Focus on: image optimization, font loading, JS bundle size

Note: This is a verify checkpoint — document findings, fix critical issues only.

### Task 4: Add sitemap.xml
Create Next.js sitemap for search engines:
- Use Next.js built-in sitemap generation
- Include all public pages: /, /about, /menu, /specialty-menu, /sushi, /bookings, /contact

File: `src/app/sitemap.ts`

### Task 5: Add robots.txt
Create robots.txt for crawler guidance:
- Allow all pages
- Reference sitemap location

File: `src/app/robots.ts`

### Task 6: Verify build and production readiness
- Run final `npm run build`
- Check for any errors or warnings
- Verify all pages render correctly
- Test that static generation works

## Verification Checkpoint
