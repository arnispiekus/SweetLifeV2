## How to reproduce the current PDF quickly

```bash
curl -s -o /tmp/menu.pdf http://localhost:3000/menu.pdf
pdftoppm -r 72 -png /tmp/menu.pdf /tmp/menu-page
magick montage /tmp/menu-page-*.png -tile 6x3 -geometry 180x254+5+5 \
  -background "#333" /tmp/menu-contact.png
```

That gives you a contact sheet of all 18 pages.
